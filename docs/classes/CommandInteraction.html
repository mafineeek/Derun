Methods

defer

  • defer(ephemeral?: boolean): Promise<void>
    • Defined in src/Structures/CommandInteraction.ts:99

    Acknowledges the interaction with a defer response. Note: You can not use more than 1 initial interaction response per interaction.

    Bot will reply with "Bot is thinking" placeholder and will wait for you to update their status. Interaction life time is 15 minutes, after it message will automatically error and your interaction token will no longer be valid. Set second param to true to make message ephermal (visible only for replied member).

    Parameters

    • Optional ephemeral: boolean

    Returns Promise<void>
